# 🚀 Custom URL Shortener API

## ✨ Features
- 🔗 Shorten long URLs
- 📊 Track click analytics (visits, referrer, etc.)
- 🔑 Google Sign-In authentication
- 🚀 High-performance caching using Redis
- ⏳ Rate limiting to prevent abuse
- 📦 Dockerized for easy deployment
